Cisim (Field) Aksiyomları

Lineer Cebir — Temel Tanımlar (Hafta 1 - Bölüm 1)

Cisim (Field): F boş olmayan bir küme olsun. Üzerinde iki işlem tanımlı: + (toplama) ve · (çarpma). Aşağıdaki 10 aksiyomu sağlıyorsa F kümesine cisim denir.

➕ Toplama Aksiyomları
F₀ Kapalılık Toplama
∀ a, b ∈ F
a + b ∈ F   ve   a · b ∈ F
Toplama ve çarpma sonucu yine F'in içinde kalır.
F₁ Birleşme (Associative) Toplama
∀ a, b, c ∈ F
a + (b + c) = (a + b) + c
Parantezin yeri fark etmez.
F₂ Birim Eleman (Etkisiz Eleman) Toplama
∀ a ∈ F, ∃ e ∈ F öyle ki:
a + e = e + a = a
Bu eleman 0 ile gösterilir. Herhangi bir cisimde toplamaya göre birim eleman sıfırdır.
F₃ Ters Eleman (Inverse) Toplama
∀ a ∈ F, ∃ b ∈ F öyle ki:
a + b = b + a = 0
Bu eleman b = −a ile gösterilir. Her elemanın toplamaya göre bir tersi vardır.
F₄ Değişme (Commutative) Toplama
∀ a, b ∈ F
a + b = b + a
Toplama sırası fark etmez.
✖️ Çarpma Aksiyomları
F₅ Birleşme (Associative) Çarpma
∀ a, b, c ∈ F
a · (b · c) = (a · b) · c
Çarpmada da parantezin yeri fark etmez.
F₆ Değişme (Commutative) Çarpma
∀ a, b ∈ F
a · b = b · a
Çarpma sırası fark etmez.
F₈ Birim Eleman (Identity) Çarpma
∃ x ∈ F \ {0} öyle ki ∀ a ∈ F:
a · x = x · a = a
Bu eleman 1 ile gösterilir. Çarpmaya göre birim elemandır. 1 ≠ 0 olmalı!
F₉ Ters Eleman (Inverse) Çarpma
∀ a ∈ F, a ≠ 0, ∃ b ∈ F öyle ki:
a · b = b · a = 1
Bu eleman b = a⁻¹ ile gösterilir. Sıfırın çarpmaya göre tersi yoktur!
🔗 Dağılma Aksiyomu
F₇ Dağılma (Distributive) Dağılma
∀ a, b, c ∈ F
a · (b + c) = a · b + a · c
Çarpma, toplamaya göre dağılır. İki işlemi birbirine bağlayan tek aksiyomdur.

Hızlı Özet

Toplama için 5 aksiyom (F₀–F₄): kapalılık, birleşme, birim eleman 0, ters eleman −a, değişme.
Çarpma için 4 aksiyom (F₅, F₆, F₈, F₉): birleşme, değişme, birim eleman 1, ters eleman a⁻¹ (a≠0).
Köprü aksiyom (F₇): dağılma kuralı — iki işlemi bağlar.
