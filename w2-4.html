Simetri Pratik & Ödev Çözümleri

Lineer Cebir — (Hafta 2 - Bölüm 4)
✏️ Pratik Sorular

Aşağıdaki matris ters-simetrik ise x, y, z değerleri ne olmalıdır?

x+y−1−11
1y+z−2−1
−11z+x−3
Çözüm — Pratik 1 Çözüm
Ters-simetrik ne demek?
AT = −A, yani Aij = −Aji tüm i,j için. Özellikle köşegen elemanları = 0 olmalı.
Köşegen koşulları (Aii = 0)
x + y − 1 = 0  →  x + y = 1

y + z − 2 = 0  →  y + z = 2

z + x − 3 = 0  →  z + x = 3
Denklem sistemi çöz
İlk ikinden: (y+z) − (x+y) = 2 − 1 → z − x = 1
Üçüncü: z + x = 3

Toplayalım: 2z = 4 → z = 2
z + x = 3 → x = 1
x + y = 1 → y = 0
Doğrulama: köşegen-dışı koşullar
A₁₂ = −1, A₂₁ = 1 → A₁₂ = −A₂₁ ✓
A₁₃ = 1, A₃₁ = −1 → A₁₃ = −A₃₁ ✓
A₂₃ = −1, A₃₂ = 1 → A₂₃ = −A₃₂ ✓
x = 1,   y = 0,   z = 2

A herhangi bir kare matris olsun. Gösterin ki:

a) A+AT, AAT ve ATA simetriktir

b) A−AT ters-simetriktir

Çözüm — Pratik 2a: A+Aᵀ simetrik İspat
Strateji: Simetrik olduğunu göstermek için transpozunu al, kendisi çıkmalı
(A + AT)T = AT + (AT)T = AT + A = A + AT

Transpoz özelliği: toplamda transpoz dağılır + çift transpoz = kendisi.

Transpozunu aldık, kendisi çıktı → A+AT simetrik. ∎
Çözüm — Pratik 2a: AAᵀ simetrik İspat
Çarpımda transpoz: sıra tersine döner
(AAT)T = (AT)T · AT = A · AT = AAT
Transpozunu aldık, kendisi çıktı → AAT simetrik. ∎
Çözüm — Pratik 2a: AᵀA simetrik İspat
(ATA)T = AT · (AT)T = AT · A = ATA
Transpozunu aldık, kendisi çıktı → ATA simetrik. ∎
Çözüm — Pratik 2b: A−Aᵀ ters-simetrik İspat
Ters-simetrik: transpoz = eksi kendisi
(A − AT)T = AT − (AT)T = AT − A = −(A − AT)
Transpozunu aldık, eksi kendisi çıktı → A−AT ters-simetrik. ∎
📝 Ödev Çözümleri

İki köşegen matrisin toplamı ve çarpımı da köşegen matristir. Bunu gösterin.

Çözüm — Ödev 1: Toplam İspat
Kurulum
A = diag(a₁, a₂, …, aₙ) ve B = diag(b₁, b₂, …, bₙ) köşegen matrisler olsun.
Yani: Aij = 0 (i≠j) ve Bij = 0 (i≠j).
i ≠ j durumu
(A + B)ij = Aij + Bij = 0 + 0 = 0

Köşegen dışı her giriş sıfır.

i = j durumu
(A + B)ii = Aii + Bii = aᵢ + bᵢ
A + B = diag(a₁+b₁, a₂+b₂, …, aₙ+bₙ) → köşegen matris. ∎
Çözüm — Ödev 1: Çarpım İspat
Genel (i,j) girişi
(A·B)ij = Σk=1n Aik · Bkj

A köşegen olduğundan Aik ≠ 0 ancak k = i ise. Yani toplamda sadece k = i terimi kalır:

(A·B)ij = Aii · Bij
i ≠ j durumu
(A·B)ij = Aii · Bij = Aii · 0 = 0

B de köşegen olduğundan Bij = 0 (i≠j). Köşegen dışı girişler sıfır.

i = j durumu
(A·B)ii = Aii · Bii = aᵢ · bᵢ
A · B = diag(a₁b₁, a₂b₂, …, aₙbₙ) → köşegen matris. ∎
Bonus: köşegen matrislerin çarpımı değişmelidir (AB = BA). Çünkü aᵢbᵢ = bᵢaᵢ.

A ve B simetrik (veya ters-simetrik) ise A+B ve cA da simetrik (veya ters-simetrik) olduğunu gösterin.

Çözüm — Ödev 2: Simetrik durum İspat
Varsayım
AT = A ve BT = B (ikisi de simetrik).
A+B simetrik mi?
(A + B)T = AT + BT = A + B

Transpozunu aldık, kendisi çıktı. ✓

cA simetrik mi?
(cA)T = c · AT = c · A = cA

Skaler transpozdan etkilenmez, AT = A kullandık. ✓

A+B ve cA simetrik. ∎
Çözüm — Ödev 2: Ters-simetrik durum İspat
Varsayım
AT = −A ve BT = −B (ikisi de ters-simetrik).
A+B ters-simetrik mi?
(A + B)T = AT + BT = (−A) + (−B) = −(A + B)

Transpoz = eksi kendisi. ✓

cA ters-simetrik mi?
(cA)T = c · AT = c · (−A) = −(cA)

Transpoz = eksi kendisi. ✓

A+B ve cA ters-simetrik. ∎
Bu ispatlar şunu söylüyor: simetrik matrisler kümesi ve ters-simetrik matrisler kümesi, toplama ve skaler çarpma altında kapalıdır (yani birer alt uzaydır — bunu ileride göreceksin).

Bu Bölümün İspat Kalıpları

"X simetrik mi?" sorusu: XT hesapla, X'in kendisi çıkıyorsa simetrik.
"X ters-simetrik mi?": XT hesapla, −X çıkıyorsa ters-simetrik.

Kullandığın araçlar her seferinde aynı:
→ (A+B)T = AT+BT   → (cA)T = cAT   → (AB)T = BTAT   → (AT)T = A

Köşegen matris ispatlarında: toplamda Aij+Bij hesapla, i≠j için sıfır olduğunu göster. Çarpımda Σ toplamında A köşegen olduğundan sadece k=i terimi sağ kalır.
