➕ 1) Matris Toplama

A ve B ikisi de n×m matris olsun. Toplamları A+B, her girişin ayrı ayrı toplanmasıyla tanımlanır:

(A + B)ij = Aij + Bij
Boyutlar aynı olmalı! Farklı boyutlu matrisler toplanamaz.

Toplama Özellikleri Özellik

∀ A, B, C ∈ Fn×m

i) Birleşme:
A + (B + C) = (A + B) + C
ii) Sıfır matris O (tüm girişleri 0):
A + O = O + A = A
iii) Değişme:
A + B = B + A
O = sıfır matris, matris toplamanın birim elemanı (nötr eleman).

Skaler Çarpma Özellikleri Özellik

∀ c, c₁, c₂ ∈ F,   ∀ A, A₁, A₂ ∈ Fn×m

i) Dağılma (matrise):
c · (A₁ + A₂) = c·A₁ + c·A₂
ii) Dağılma (skalere):
(c₁ + c₂) · A = c₁·A + c₂·A
iii) Birleşme:
(c₁ · c₂) · A = c₁ · (c₂ · A)
iv) Birim & Ters:
1 · A = A     (−1) · A = −A
Sonuç: A − A = A + (−A) = O   ve   A − B = A + (−B) olarak tanımlanır.
🔗 3) Matris Çarpımı

A bir n×m, B bir m×r matris olsun. A·B çarpımı n×r boyutunda bir matristir:

(A·B)ij = Σk=1m Aik · Bkj = Ai1B1j + Ai2B2j + ⋯ + AimBmj
A'nın i. satırı ile B'nin j. sütununu eleman eleman çarp, topla → sonuç matrisin (i,j) girişi.

Matris Çarpımı Özellikleri Özellik

Uygun boyutlu A, B, C matrisleri ve c skaleri için:

i) Birleşme:
(A·B)·C = A·(B·C)
ii) Dağılma:
A·(B + C) = A·B + A·C
iii) Birim matris:
I·A = A     A·I = A
iv) Skaler uyum:
c·(A·B) = (c·A)·B = A·(c·B)

Matris Kuvveti Gösterim

A kare matris ise kuvvetleri tanımlıdır:

A² = A·A     A³ = A·A² = A²·A     ⋯
Sadece kare matrislerde kuvvet tanımlıdır (boyut uyumu gerektiğinden).

Hızlı Özet

Toplama: Aynı boyutlu matrislerde giriş giriş topla. Değişmeli ve birleşmeli. Nötr eleman = O (sıfır matris).
Skaler çarpma: Her girişi c ile çarp. Dağılma, birleşme, birim (1·A=A) özellikleri var. (−1)·A = −A.
Matris çarpımı: A'nın satırı × B'nin sütunu → toplam. An×m · Bm×r = Cn×r. A'nın sütun sayısı = B'nin satır sayısı şart!
⚠️ Değişmeli DEĞİL: AB ≠ BA genel olarak. Bu lineer cebirin en temel farkı!
