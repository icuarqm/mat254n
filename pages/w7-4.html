✏️ Soru 2 — $\mathbb{R}^3$ için İki Baz

Aşağıdaki kümelerin $\mathbb{R}^3$'ün bazı olduğunu gösteriniz:

·  $S = \{(1, 1, 1),\; (1, -1, 1),\; (2, 0, 1)\}$
·  $T = \{(1, 1, 1),\; (1, 1, 0),\; (1, 0, 0)\}$
Çözüm Çözüm

$\dim \mathbb{R}^3 = 3$ ve her iki küme de $3$ vektör içerdiğinden, bağımsızlık ya da üreticilik şartlarından sadece birini göstermek yeterlidir (önceki haftadan sonuç).

$S$ için bağımsızlık

$c_1 (1, 1, 1) + c_2 (1, -1, 1) + c_3 (2, 0, 1) = (0, 0, 0)$:

$$\begin{aligned} c_1 + c_2 + 2 c_3 &= 0 \\ c_1 - c_2 \phantom{ + 2 c_3 } &= 0 \\ c_1 + c_2 + c_3 &= 0 \end{aligned}$$

Egzersiz olarak çözüldüğünde tek çözüm $c_1 = c_2 = c_3 = 0$ → $S$ lineer bağımsız → $S$ bir bazdır.

$T$ için üreticilik

Keyfi $(a, b, c) \in \mathbb{R}^3$ için:

$$(a, b, c) = c_1 (1, 1, 1) + c_2 (1, 1, 0) + c_3 (1, 0, 0) = (c_1 + c_2 + c_3,\; c_1 + c_2,\; c_1)$$

Doğrudan okuma ile:

$$c_1 = c, \qquad c_2 = b - c, \qquad c_3 = a - b.$$

Her $(a, b, c)$ için çözüm var → $T$ $\mathbb{R}^3$'ü üretir → $T$ bir bazdır.

Çözüm — 2. Yol (Yapılandırmacı Yöntem) Çözüm

Boyutu artırarak bağımsız vektörler ekleyelim. Adımda her seferinde, mevcut span'in dışından bir vektör seçiyoruz.

Adım 1 — $W_1 = \langle (1, 2, -3, 0) \rangle$

$W_1 \subsetneq \mathbb{R}^4$. Mesela $w_1 = (0, 0, 0, 1) \notin W_1$ (4. bileşeni nonzero, $W_1$'de değil).

$$W_2 = \langle (1, 2, -3, 0),\; (0, 0, 0, 1) \rangle, \qquad \dim W_2 = 2.$$
Adım 2 — $W_2 \subsetneq \mathbb{R}^4$

$w_2 \in \mathbb{R}^4 \setminus W_2$ seçmek için: $w_2 \neq c_1 (1, 2, -3, 0) + c_2 (0, 0, 0, 1)$ olmalı. Mesela $w_2 = (0, 1, 1, 0)$ uygundur.

$$W_3 = \langle (1, 2, -3, 0),\; (0, 0, 0, 1),\; (0, 1, 1, 0) \rangle, \qquad \dim W_3 = 3.$$
Adım 3 — $W_3 \subsetneq \mathbb{R}^4$

$W_3$'ün genel elemanı $c_1(1, 2, -3, 0) + c_2(0, 0, 0, 1) + c_3(0, 1, 1, 0) = (c_1,\; 2c_1 + c_3,\; -3c_1 + c_3,\; c_2)$. Bu biçimde olmayan bir $w_3$ seç → $w_3 = (1, -1, -1, 0)$ (2. ve 3. bileşenler $c_1$ ve $c_3$ ilişkisine uymaz).

$$W_4 = \langle (1, 2, -3, 0),\; (0, 0, 0, 1),\; (0, 1, 1, 0),\; (1, -1, -1, 0) \rangle.$$

$\dim W_4 = 4$ ve $W_4 \subseteq \mathbb{R}^4$ → $W_4 = \mathbb{R}^4$.

$$\beta = \{(1, 2, -3, 0),\; (0, 0, 0, 1),\; (0, 1, 1, 0),\; (1, -1, -1, 0)\}$$
