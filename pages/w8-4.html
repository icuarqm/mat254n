Parametrik çözüm

Serbest değişkenler: $c = t,\; d = k$. İndirgenmiş sistemden:

$$5 b - 2 c - 3 d = 0 \;\Rightarrow\; b = \tfrac{2 t + 3 k}{5}, \qquad a = b - c - d = \tfrac{-3 t - 2 k}{5}.$$
$$(a, b, c, d) = t \left(-\tfrac{3}{5}, \tfrac{2}{5}, 1, 0\right) + k \left(-\tfrac{2}{5}, \tfrac{3}{5}, 0, 1\right).$$

Parametrik çözüm

$2$ pivot, $4$ bilinmeyen → $2$ serbest değişken: $z$ ve $t$.

İkinci satırdan: $6 y - 46 z - 6 t = 0 \;\Rightarrow\; y = \tfrac{23 z + 3 t}{3}$.

Birinciden: $x + y - 10 z - t = 0 \;\Rightarrow\; x = -y + 10 z + t = \tfrac{7 z}{3}$.

$$(x, y, z, t) = z \left(\tfrac{7}{3}, \tfrac{23}{3}, 1, 0\right) + t \left(0, 1, 0, 1\right).$$

$z = 3 s$ alıp tam sayılı baz da yazılabilir: $(7, 23, 3, 0)$.

$$\text{Çözüm uzayı bazı:} \;\; \{\, (7, 23, 3, 0),\;\; (0, 1, 0, 1)\, \}, \qquad \dim = 2.$$
