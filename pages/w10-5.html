Lineer Cebir — (Hafta 10 - Bölüm 5)
✏️ Egzersiz 1 — Lineer Dönüşüm mü?

Aşağıdaki fonksiyonların lineer dönüşüm olup olmadığını gösteriniz:

a)  $L : \mathbb{R}^{2 \times 2} \to \mathbb{R}^2,\; \begin{bmatrix}a&b\\c&d\end{bmatrix} \mapsto (a + d,\; b + c)$
b)  $L : \mathbb{R}^{n \times n} \to \mathbb{R},\; A \mapsto \operatorname{tr}(A)$
c)  $L : \mathbb{R}^4 \to \mathbb{R}^3,\; (x_1, x_2, x_3, x_4) \mapsto (x_1 + x_2,\; x_3 + x_4,\; 1)$
d)  $L : \mathbb{R}^2 \to \mathbb{R}^2,\; (x_1, x_2) \mapsto (x_1 \cdot x_2,\; -x_1)$
Çözüm a Lineer Dönüşüm

Tüm bileşenler matris girdilerinin lineer kombinasyonu olduğundan toplama ve skaler katlama altında korunur:

$$L(A + B) = ((a + a') + (d + d'),\; (b + b') + (c + c')) = L(A) + L(B), \qquad L(k A) = (k a + k d,\; k b + k c) = k \cdot L(A).$$
Lineer dönüşümdür.
Çözüm b Lineer Dönüşüm

İz fonksiyonu köşegen elemanların toplamıdır → girdilerde lineer:

$$\operatorname{tr}(A + B) = \operatorname{tr}(A) + \operatorname{tr}(B), \qquad \operatorname{tr}(c A) = c \cdot \operatorname{tr}(A).$$
Lineer dönüşümdür.
Çözüm c Lineer dönüşüm değil

Sıfır vektörünün görüntüsü sıfır olmalıydı:

$$L(0, 0, 0, 0) = (0, 0, 1) \neq (0, 0, 0).$$

Sabit terim "$1$" toplamsallığı ve homojenliği bozar → lineer değildir.

Çözüm d Lineer dönüşüm değil

Karşı-örnek: $u = v = (1, 1) \Rightarrow u + v = (2, 2)$.

$$L(u) + L(v) = (1, -1) + (1, -1) = (2, -2), \qquad L(u + v) = L(2, 2) = (4, -2).$$

$x_1 \cdot x_2$ terimi kuadratik → lineer değildir.

✏️ Egzersiz 2 — Karma Polinom Operatörü

$L : \mathbb{R}[x] \to \mathbb{R}[x],\; L(p) = x^2 \cdot p'(x) + x \cdot \displaystyle\int_0^1 p(t)\, dt$ dönüşümünün lineer olduğunu gösteriniz.

Çözüm Çözüm

$a, b \in \mathbb{R},\; p_1, p_2 \in \mathbb{R}[x]$ için tek-adım testi:

$$L(a p_1 + b p_2) = x^2 (a p_1 + b p_2)' + x \int_0^1 (a p_1 + b p_2)\, dt$$
$$= x^2 (a p_1' + b p_2') + x \left( a \int_0^1 p_1\, dt + b \int_0^1 p_2\, dt \right)$$
$$= a \left( x^2 p_1' + x \int_0^1 p_1\, dt \right) + b \left( x^2 p_2' + x \int_0^1 p_2\, dt \right) = a L(p_1) + b L(p_2). \quad \checkmark$$

Hem türev hem integral birer lineer işlem; bunların skalerle ölçeklenmiş toplamı da lineerdir.

✏️ Egzersiz 4 — $\dim \operatorname{Ker} L$ ve $\dim \operatorname{im} L$

$L : \mathbb{R}^{2 \times 2} \to \mathbb{R}^3,\; L\!\begin{bmatrix}a&b\\c&d\end{bmatrix} = (a + b + c,\; b + c + d,\; a + d)$ verilsin. $\dim \operatorname{Ker} L$ ve $\dim \operatorname{im} L$ değerlerini bulunuz.

Çözüm — Çekirdek Çözüm

$L(\cdot) = (0, 0, 0)$:

$$a + b + c = 0, \quad b + c + d = 0, \quad a + d = 0.$$

3. denklemden $d = -a$. 2. denklemde: $b + c - a = 0 \Rightarrow b + c = a$. 1. denkleme yerleştir: $a + a = 0 \Rightarrow a = 0$ → $d = 0$ ve $c = -b$.

$b = t$ alınca: $(a, b, c, d) = (0, t, -t, 0) = t \cdot \begin{bmatrix} 0 & 1 \\ -1 & 0 \end{bmatrix}$.

$\operatorname{Ker} L = \left\langle \begin{bmatrix} 0 & 1 \\ -1 & 0 \end{bmatrix} \right\rangle$, $\dim \operatorname{Ker} L = 1$.
Çözüm — Görüntü Çözüm

$L$'yi standart bazda ayır:

$$L = a (1, 0, 1) + b (1, 1, 0) + c (1, 1, 0) + d (0, 1, 1).$$

$b$ ve $c$'nin görüntüleri aynı → $\operatorname{im} L = \langle (1, 0, 1),\; (1, 1, 0),\; (0, 1, 1) \rangle$.

Bağımsızlık: $a(1,0,1) + b(1,1,0) + c(0,1,1) = (a+b, b+c, a+c) = 0$ sistemini çözünce tek çözüm $a = b = c = 0$ → 3 vektör bağımsız.

$\dim \operatorname{im} L = 3$.

Rank–Nullity: $\dim \mathbb{R}^{2 \times 2} = 4 = 1 + 3$ ✓

✏️ Egzersiz 6 — $W_1 \cong W_2$

$W_1$: aşağıdaki sistemin çözüm uzayı:

$$x_1 + x_2 - x_3 + x_4 = 0, \qquad x_1 + x_2 + x_3 - x_4 = 0.$$

$W_2$:  $\langle 1 + x + x^2 + x^3,\;\; x - x^3,\;\; 1 - x + x^2 + 3 x^3 \rangle$.

$W_1 \cong W_2$ olduğunu gösteriniz.

Parametrik çözüm

2. satır: $2 x_3 - 2 x_4 = 0 \Rightarrow x_3 = x_4$. 1. satır: $x_1 = -x_2 + x_3 - x_4 = -x_2$.

$x_2 = a,\; x_4 = b$:  $(x_1, x_2, x_3, x_4) = (-a, a, b, b) = a (-1, 1, 0, 0) + b (0, 0, 1, 1)$.

$W_1 = \langle (-1, 1, 0, 0),\; (0, 0, 1, 1) \rangle$,  $\dim W_1 = 2$.
Çözüm — $W_2$'nin Boyutu Çözüm
$\{1, x, x^2, x^3\}$ bazında koordinatla
·  $p_1 = 1 + x + x^2 + x^3 \;\longleftrightarrow\; (1, 1, 1, 1)$
·  $p_2 = x - x^3 \;\longleftrightarrow\; (0, 1, 0, -1)$
·  $p_3 = 1 - x + x^2 + 3 x^3 \;\longleftrightarrow\; (1, -1, 1, 3)$
Bağımlılık kontrolü

$p_3 \stackrel{?}{=} \alpha p_1 + \beta p_2 = (\alpha,\; \alpha + \beta,\; \alpha,\; \alpha - \beta)$. Sırasıyla:

$\alpha = 1,\; \alpha + \beta = -1 \Rightarrow \beta = -2,\; \alpha = 1$ ✓, $\alpha - \beta = 1 - (-2) = 3$ ✓.

Yani $p_3 = p_1 - 2 p_2$ → $p_3$ bağımlı.

Geriye kalan iki polinom bağımsız

$(1, 1, 1, 1)$ ve $(0, 1, 0, -1)$ açıkça birbirinin katı değil.

$W_2 = \langle p_1,\; p_2 \rangle = \langle 1 + x + x^2 + x^3,\; x - x^3 \rangle$,  $\dim W_2 = 2$.
Sonuç — İzomorfizm İspat Bitti

$\dim W_1 = 2 = \dim W_2$ ve her ikisi de sonlu boyutlu vektör uzaylarıdır. İzomorfizm teoremi gereği:

$$W_1 \;\cong\; W_2. \qquad \blacksquare$$
