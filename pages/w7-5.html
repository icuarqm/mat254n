✏️ Soru 7 — Bağımsız Vektörlerin Kombinasyonu

$v_1, v_2, v_3$ lineer bağımsız vektörler olsun. $\{v_1 + v_2,\;\; v_1 + v_2 - v_3,\;\; v_1 + 2 v_2 + 3 v_3\}$ kümesinin lineer bağımsız olup olmadığını inceleyiniz.

Çözüm Çözüm
Eşitliği aç
$$c_1 (v_1 + v_2) + c_2 (v_1 + v_2 - v_3) + c_3 (v_1 + 2 v_2 + 3 v_3) = \mathbf{0}$$

$v_1, v_2, v_3$'ün katsayılarını toplayarak:

$$(c_1 + c_2 + c_3)\, v_1 \;+\; (c_1 + c_2 + 2 c_3)\, v_2 \;+\; (-c_2 + 3 c_3)\, v_3 \;=\; \mathbf{0}$$
Bağımsızlığı kullan

$v_1, v_2, v_3$ lineer bağımsız olduğundan her katsayı $0$ olmalı:

$$\begin{aligned} c_1 + c_2 + c_3 \phantom{ + 0 } &= 0 \\ c_1 + c_2 + 2 c_3 &= 0 \\ \phantom{ c_1 + {} } -c_2 + 3 c_3 &= 0 \end{aligned}$$

Parametrik çözüm

İndirgenmiş sistem:

$$a - b + c + d = 0, \qquad 5 b - 2 c - 3 d = 0.$$

$c$ ve $d$ serbest değişken olsun: $c = t,\; d = k$. O zaman

$$b = \tfrac{2 t + 3 k}{5}, \qquad a = b - c - d = \tfrac{-3 t - 2 k}{5}.$$

$(a, b, c, d)$'yi $t$ ve $k$ cinsinden ayır:

$$(a, b, c, d) = t \left(-\tfrac{3}{5},\; \tfrac{2}{5},\; 1,\; 0\right) + k \left(-\tfrac{2}{5},\; \tfrac{3}{5},\; 0,\; 1\right).$$
Span yapısı

Her $W$ elemanı bu iki vektörün lineer kombinasyonu → $W$ bu iki vektörle üretilir. Üstelik bu iki vektör lineer bağımsızdır (3. ve 4. bileşenlere bakıldığında $(1, 0)$ ve $(0, 1)$ birbirinin katı değil).

$$W = \left\langle \left(-\tfrac{3}{5},\; \tfrac{2}{5},\; 1,\; 0\right),\; \left(-\tfrac{2}{5},\; \tfrac{3}{5},\; 0,\; 1\right) \right\rangle, \qquad \dim W = 2.$$
