✅ Soru 6: A Tersinir İken İspat

A tersinir olduğunda Soru 5'teki ifadeleri ispatlayın:

a) AB = O ⟹ B = O

b) AB = AC ⟹ B = C

c) AB = A ⟹ B = I

Çözüm 6a — AB = O, A tersinir ⟹ B = O İspat
AB = O

İki tarafı soldan A⁻¹ ile çarp:

A⁻¹(AB) = A⁻¹ · O
(A⁻¹A)B = O
I · B = O
B = O
A tersinir ve AB = O ⟹ B = O. ∎
Çözüm 6b — AB = AC, A tersinir ⟹ B = C İspat
AB = AC

İki tarafı soldan A⁻¹ ile çarp:

A⁻¹(AB) = A⁻¹(AC)
(A⁻¹A)B = (A⁻¹A)C
B = C
A tersinir ve AB = AC ⟹ B = C. ∎
Çözüm 6c — AB = A, A tersinir ⟹ B = I İspat
AB = A

İki tarafı soldan A⁻¹ ile çarp:

A⁻¹(AB) = A⁻¹A
(A⁻¹A)B = I
B = I
A tersinir ve AB = A ⟹ B = I. ∎

Bu Bölümün Dersleri

RREF + elementer matrisler: R = Eₖ···E₁·A. Her ERO adımı bir Eᵢ matrisi.

Elementer çarpım olarak yazma: A ∼ I ise E···E₁·A = I → A = E₁⁻¹···Eₖ⁻¹ (elementer matrislerin çarpımı).

Tersinirlik testi: [A:I] → ERO. Sıfır satır → tekil. I çıkarsa → tersinir, sağ taraf = A⁻¹.

Parametreli matris: Echelon formda pivotların sıfır olduğu durumları bul → o parametrede tekil.

Kritik fark: AB = O, AB = AC, AB = A gibi ifadeler genel olarak "sadeleştirilemez" — A'nın tersinir olması şart. Tersinir değilse karşı örnekler mevcut.
